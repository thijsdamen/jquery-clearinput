jQuery Clear Input by thijsdamen

Plugin created to easily facilitate placeholders in form friends.

About this plugin -------------------------------------- This plugin was created to clearn input fields when focused. It is custom these days to use a placeholder value in an input field. And for every single you you needed 3 or 4 lines of code. No more, mostly to suit my own needs and hopefully someone else's this plugin was created. Goals of the plugin -------------------------------------- Make it very easy to use. Functionality -------------------------------------- - Empty a form field when focused (if placeholder value is still entered) - Repopulate form field if blurred (if no value was entered) - Submit empty values (if placeholder value is still entered) - Submit custom values (if placeholder value is still entered)

Dependencies

[jQuery](http://jquery.com/) 1.4.4+
